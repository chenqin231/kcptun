# 故障处理规范

> **触发信号**：P0-P3 故障、紧急回滚、生产问题

---

## 故障等级

| 等级 | 描述 | 响应时间 | 处理方式 |
|------|------|----------|----------|
| P0 | 严重故障（服务不可用） | 立即响应 | 立即回滚或热修复 |
| P1 | 重要故障（核心功能异常） | 30分钟内 | 紧急修复 |
| P2 | 一般故障（部分功能异常） | 2小时内 | 计划修复 |
| P3 | 轻微问题（体验问题） | 1天内 | 正常修复 |

---

## 故障处理流程

```
故障发生
    ↓
立即评估（等级判定）
    ├─ P0/P1 → 立即响应
    │     ├─ 回滚到稳定版本
    │     └─ 启动紧急修复
    └─ P2/P3 → 正常流程
          ├─ 创建 Issue
          ├─ 定位根因
          └─ 计划修复
    ↓
修复验证
    ↓
复盘总结
```
